Live Visuals for Nam June Paik: I Expose the Music. vernissage.

Museum Ostwall - Dortmunder U. Dortmund, Germany

The idea behind this project is to combine digital and analogue technologies to create a hybrid visual performance/installation. Using analogue and digital video cameras, an oscilloscope and a laptop, input signals are switched on and off, creating various feedback loops. As a visual reference to Nam June Paik's work and analogue video synthesis, the project was created as part of the opening of the Paik exhibition at the Museum Ostwall, where a similar setup was used for live visualisations. All media are interconnected and linked via live coding, with the code for image generation being written in real time. In the future, additional oscilloscopes will be connected to create a multimedia art installation. Algorithms are increasingly determining our everyday lives. The intention of this installation is to use them for creative and non-normative use of media, as Paik did decades ago with analogue technology such as televisions and cameras.
